# kpop-social-media-back-end

A api se trata de uma "rede social" para quem gosta de Kpop, você vai poder se cadastrar e logar, usuários poderão favoritar grupos musicais e artistas de Kpop. Artistas tem pontuações que vão de 0 até 10 para cada parte técnica, além de poderem serem incluídos em algum grupo musical.
